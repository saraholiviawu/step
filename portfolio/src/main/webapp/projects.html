What's Cooking?


Designed a web application that promotes food sustainability and minimizes food waste by allowing users to search for food recipes based on available ingredients.


With each query, this program calls a recipe search API, parses through its database for the given ingredients, and returns the corresponding recipe options. Programmed in HTML, CSS, JavaScript, and Python.

Immune Cell and Bacteria Model


Modeled the migration of a neutrophil immune cell with subcellular protein diffusion and reaction using Matlab. Allows for a visual analysis of the effect of a chemoatttractant signal emitted by E. Coli bacteria on the immune cell.
